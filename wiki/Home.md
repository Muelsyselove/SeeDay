## 📖 项目概述

**SeeDay（顾日）** 是一个优雅的个人时间洞察仪表盘，帮助你追踪和理解日常的数字活动。通过轻量级的本地代理程序收集应用使用数据，并以美观的可视化界面呈现你的时间分布。

> 💡 本项目基于 [live-dashboard](https://github.com/Monika-Dream/live-dashboard) 进行二次开发，感谢原作者的开源贡献。

## ✨ 核心特性快速概览

| 特性 | 说明 |
|------|------|
| 🕐 实时活动追踪 | 跨平台 Agent 自动检测当前活动应用，实时更新状态 |
| 📊 可视化仪表盘 | 精美的 UI，支持多种主题风格（默认/水墨国风/个性化定制） |
| 🤖 AI 每日总结 | 每晚自动生成当日活动总结，支持多种 AI 提供商 |
| 🔒 隐私保护 | 提供应用分级过滤和敏感词过滤机制 |
| 🐳 Docker 部署 | 一键容器化部署，简化运维流程 |
| 🎨 主题系统 | 支持多种主题风格切换，满足个性化需求 |
